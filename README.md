# Tools
* create-react-native-app cli
* native-base
* react-navigation
* react-native-image-picker
* react-native-simple-radio-button

# Start App

Install CRNA globally, Clone repo, install, cd into folder and run:
```git
npm install -g create-react-native-app
npm install
npm run ios
```
